前端库展示

这里展示该站使用的库,仅供大家参考

jQuery

jQuery 是一个快速、小巧、功能丰富的 JavaScript 库，适用于各种浏览器。

官网：https://jquery.com/

React

React 是一个用于构建用户界面的 JavaScript 库，由 Facebook 开发并维护。

官网：https://reactjs.org/

jszip

jszip 是一个用于创建、读取和修改 ZIP 文件的 JavaScript 库。

官网：https://stuk.github.io/jszip/

Vue

Vue 是一个渐进式 JavaScript 框架，用于构建用户界面。

官网：https://cn.vuejs.org/

ffmpeg.js

ffmpeg.js 是一个运行在浏览器中的媒体处理库，提供了视频和音频的编解码、剪切、合并等功能。

官网：https://ffmpegwasm.github.io/

Math.js

Math.js 是一个用于进行数学计算和表达式求值的 JavaScript 库。

官网：https://mathjs.org/

Algebra.js

Algebra.js 是一个用于进行代数计算和符号运算的 JavaScript 库。

官网：http://algebra.js.org/

Bootstrap

Bootstrap 是一个用于开发响应式布局、移动设备优先的前端开发框架。它包含了HTML、CSS及JS等各种组件，可帮助快速搭建网站或Web应用。

官网：https://getbootstrap.com/
